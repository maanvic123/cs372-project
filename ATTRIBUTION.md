# ATTRIBUTION.md

## 1. Data Sources
### Spotify Track Dataset
The raw dataset with 1 million tracks and audio features is from Kaggle: https://www.kaggle.com/datasets/amitanshjoshi/spotify-1million-tracks.


## 2. External APIs
### OpenAI API
Used to generate text embeddings with text-embedding-3-small model. Documentation: https://platform.openai.com/docs/api-reference.

### Spotify Web API
Used for generating Spotify song embeddings and token access. Documentation: https://developer.spotify.com/documentation/web-api.


## 3. Libraries & Tools
The following open-source libraries were used (all declared in `requirements.txt`):
- **FAISS (faiss-cpu)** — Facebook AI Research  
  High-performance vector search backend.  
  https://github.com/facebookresearch/faiss  

- **Streamlit** — interactive UI framework  
  https://streamlit.io  

- **NumPy** — numerical computation  
  https://numpy.org/  

- **Pandas** — data loading and preprocessing  
  https://pandas.pydata.org/  

- **python-dotenv** — environment variable loading  
  https://github.com/theskumar/python-dotenv  

- **Requests** — HTTP requests for Spotify’s API  
  https://docs.python-requests.org/
